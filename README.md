## Description

It is a chat on Nest.js(Passport(JWT+Local)), Socket.io and MongoDB(Mongoose). You can:
1. Login, Register
2. Create room, Read room by user_id.
3. Create message, Read messages by room_id.

See OpenAPI(Swagger) http://localhost:3000/api/#/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
